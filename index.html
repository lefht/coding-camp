Session 1 — Getting started

Students set up Python, learn the basics, and write their first programs.

print() variables data types input() if / else

Core concepts

print()Displays output to the screen. Example: print("Hello!")
VariablesStore data. name = "Alice", age = 16, score = 9.5
Data typesStrings, integers, floats, booleans — the kinds of values Python knows about.
input()Asks the user to type something. Always returns a string.
if / elif / elseMake decisions in code based on conditions.

Problems

Problem 1 — Hello, you!

Ask the user for their name, then print a greeting. Example output: Hello, Maya!

name = input("What is your name? ")
print("Hello, " + name + "!")

Problem 2 — Age checker

Ask the user for their age. Print "You can vote!" if they are 18 or older, otherwise print "Not old enough yet."

age = int(input("How old are you? "))
if age >= 18:
    print("You can vote!")
else:
    print("Not old enough yet.")

Problem 3 — Simple calculator

Ask for two numbers, then print their sum, difference, product, and quotient.

a = float(input("First number: "))
b = float(input("Second number: "))
print("Sum:", a + b)
print("Difference:", a - b)
print("Product:", a * b)
if b != 0:
    print("Quotient:", a / b)
else:
    print("Cannot divide by zero!")

Problem 4 — Grade classifier

Ask for a score (0–100). Print "A" for 90+, "B" for 80+, "C" for 70+, "D" for 60+, or "F" otherwise.

score = int(input("Enter score: "))
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:

Session 2 — Loops and lists

Students repeat actions with loops and work with collections of data.

for loops while loops lists range() len()

Core concepts

for loopRepeats code for each item. for i in range(5): runs the block 5 times.
while loopKeeps going as long as a condition is true. while count < 10:
ListsA collection of items. fruits = ["apple", "banana", "cherry"]
range()Generates a sequence of numbers. range(1, 6) gives 1, 2, 3, 4, 5.
append / lenAdd items with .append(), count items with len().

Problems

Problem 1 — Count to N

Ask the user for a number N. Print every number from 1 to N.

n = int(input("Enter a number: "))
for i in range(1, n + 1):
    print(i)

Problem 2 — Sum of a list

Create a list of 5 numbers. Use a loop to add them all up and print the total.

numbers = [4, 7, 2, 9, 1]
total = 0
for num in numbers:
    total = total + num
print("Total:", total)

Problem 3 — Guessing game

Pick a secret number (e.g. 7). Use a while loop to keep asking the user to guess until they get it right. Tell them if each guess is too high or too low.

secret = 7
guess = 0
while guess != secret:
    guess = int(input("Guess the number: "))
    if guess < secret:
        print("Too low!")
    elif guess > secret:
        print("Too high!")
print("Correct! The number was", secret)

Problem 4 — Shopping list

Let the user keep adding items to a list until they type "done". Then print all the items.

shopping = []
while True:
    item = input("Add item (or 'done'): ")
    if item == "done":
        break
    shopping.append(item)

print("Your list:")
for item in shopping:
    print("-", item)

Session 3 — Functions

Students learn to write reusable blocks of code that take inputs and return outputs.

def parameters return scope built-ins

Core concepts

defDefine a function: def greet(name): — give it a name and any inputs it needs.
ParametersThe variables a function receives. def add(a, b): has two parameters.
returnSends a value back. return a + b gives the answer to whoever called the function.
ScopeVariables inside a function are local — they don't exist outside it.
Built-insmax(), min(), round(), abs() — functions Python gives you for free.

Problems

Problem 1 — Greeting function

Write a function called greet that takes a name and prints "Hello, [name]!". Call it three times with different names.

def greet(name):
    print("Hello, " + name + "!")

greet("Alice")
greet("Bob")
greet("Carlos")

Problem 2 — Area calculator

Write a function area that takes width and height and returns the area of a rectangle. Test it with a few different sizes.

def area(width, height):
    return width * height

print(area(4, 5))
print(area(10, 3))
print(area(7, 7))

Problem 3 — Is it even?

Write a function is_even that takes a number and returns True if it's even, False if not. Test it with several numbers.

def is_even(number):
    return number % 2 == 0

print(is_even(4))
print(is_even(7))
print(is_even(0))

Problem 4 — Biggest of three

Write a function biggest that takes three numbers and returns the largest one. Do not use Python's built-in max().

def biggest(a, b, c):
    result = a
    if b > result:
        result = b
    if c > result:
        result = c
    return result

print(biggest(3, 7, 5))
print(biggest(10, 2, 8))

Problem 3 — Mini quiz game

Write a quiz with 3 questions. Keep track of the score and print the final result at the end.

score = 0

answer = input("What is the capital of France? ").strip().lower()
if answer == "paris":
    print("Correct!")
    score += 1
else:
    print("Wrong! It's Paris.")

answer = input("What is 7 x 8? ").strip()
if answer == "56":
    print("Correct!")
    score += 1
else:
    print("Wrong! It's 56.")

answer = input("What language are we learning? ").strip().lower()
if answer == "python":
    print("Correct!")
    score += 1
else:
    print("Wrong! It's Python.")

print("Your score:", score, "out of 3")

Problem 4 — Final project: number stats

Let the user enter numbers one by one (stop when they type "done"). Then print the count, total, average, highest, and lowest number entered.

numbers = []

while True:
    entry = input("Enter a number (or 'done'): ")
    if entry == "done":
        break
    numbers.append(float(entry))

if len(numbers) == 0:
    print("No numbers entered.")
else:
    print("Count:", len(numbers))
    print("Total:", sum(numbers))
    print("Average:", sum(numbers) / len(numbers))
    print("Highest:", max(numbers))
    print("Lowest:", min(numbers))
