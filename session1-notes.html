📓 Session 1 Notes

Getting started — visuals & examples

1 — print()
print("Hello, world!") print("I am", 10, "years old")

Think of print() as your code talking out loud. The text appears instantly in the output window.

2 — variables
name = "Sofia" age = 11 favourite_colour = "blue"

A variable is like a sticky-note box. The label on the front is the name; the value inside is what Python stores.

3 — data types
player_name = "Kai" # str score = 150 # int speed = 4.5 # float game_over = False # bool

Each type has its own purpose. Strings need quotes; numbers and booleans never do. Mixing them without converting causes an error!

4 — input()
name = input("What is your name? ") print("Hello,", name)

input() always hands back a string — even if the user types a number! Use int() to convert it when you need to do maths.

5 — if / else
score = int(input("Enter your score: ")) if score >= 90: print("Grade: A") elif score >= 70: print("Grade: B") else: print("Grade: C")

Python checks conditions from top to bottom and takes the first one that is True. Once a branch runs, it skips the rest. Draw this on the board before showing the code — the flowchart makes the structure obvious.
