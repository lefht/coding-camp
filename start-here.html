🚀 Start Here!

Set up your computer before the first session — it only takes a few minutes!

🐍 Step 1

Install Python

Python is the programming language we'll be using. You need to install it so your computer understands the code you write.

  1. 1Click the download button below — it takes you to the official Python website.
  2. 2Click the big yellow "Download Python" button on that page.
  3. 3Open the file that downloads (it ends in .exe on Windows or .pkg on Mac).
  4. 4⚠️ Important on Windows: before clicking Install, tick the box that says "Add Python to PATH" at the bottom of the screen!
  5. 5Click Install Now and wait for it to finish.
⬇️ Download Python
💡 How to check it worked: Open a terminal (search "Terminal" on Mac or "Command Prompt" on Windows) and type python --version then press Enter. You should see something like Python 3.x.x.
💻 Step 2

Install VS Code

VS Code (Visual Studio Code) is where you'll write your code. Think of it as a really smart notepad made just for programmers — it highlights your code in colours, spots mistakes, and makes everything easier to read.

  1. 1Click the download button below.
  2. 2The website will detect your operating system automatically — click the big blue download button.
  3. 3Open the downloaded file and follow the installer steps. The default options are all fine — just keep clicking Next.
  4. 4Once installed, open VS Code. You should see a welcome screen.
⬇️ Download VS Code
🧩 Step 3

Add the Python Extension to VS Code

VS Code works with add-ons called extensions. The Python extension adds colour-coding and other helpful tools for Python.

  1. 1Open VS Code.
  2. 2Click the Extensions icon on the left sidebar — it looks like four squares 🧩.
  3. 3In the search bar at the top, type Python.
  4. 4Click the one made by Microsoft (it should be the first result).
  5. 5Click the blue Install button.
💡 Tip: You only need to do this once — the extension stays installed forever.
✅ Step 4

Test That Everything Works

Let's make sure Python and VS Code are talking to each other properly.

  1. 1In VS Code, click File → New File, then save it as test.py (the .py ending tells VS Code it's a Python file).
  2. 2Type this into the file: print("Hello, world!")
  3. 3Click the ▶ Run button (top right corner) or press Ctrl + F5 (Windows) / Cmd + F5 (Mac).
  4. 4A panel should appear at the bottom showing: Hello, world!
🎉 If you see "Hello, world!" — you're all set! Your computer is ready for the first session. See you there!
📋 Quick Checklist

Before Session 1, make sure you have:

  • 🐍 Python installed
  • 💻 VS Code installed
  • 🧩 Python extension added to VS Code
  • ✅ Ran print("Hello, world!") successfully
